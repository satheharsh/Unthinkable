# UI Guidelines

## Principles
- **Modern & Professional:** Clean layouts tailored for healthcare professionals and patients.
- **Accessible:** Semantic HTML, ARIA labels, and keyboard navigation.

## Component Library
- **Core:** Shadcn UI
- **Styling:** Tailwind CSS
- **Motion:** Framer Motion for micro-interactions (e.g., slot selection, modal transitions)

## Color Palette
- Needs to be strictly adhered to using Tailwind utility classes based on the Shadcn UI theme configuration.
